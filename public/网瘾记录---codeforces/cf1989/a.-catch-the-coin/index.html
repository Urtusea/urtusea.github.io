    A. Catch the Coin

    前言： #

    • Links: A. Catch the Coin
    • Codeforces: 待更新
    • Clist: \(784\)
    • Tags: 模拟
    • 时间复杂度: \(O(1)\)

    题意： #

    给你一个硬币的坐标 \((x, y)\) ，你初始位置在 \((0, 0)\) ，对于每一时刻均可以向满足 \(|x'-x| + |y'-y| \le 2\) 的点 \((x', y')\) 移动，硬币在你移动后坐标变为 \((x, y - 1)\) ，问你是否有一时刻可以接到硬币

    思路： #

    观察可以发现，只有当 \(y \geq -1\) 时才满足条件

    代码： # 

    #include <bits/stdc++.h>
using namespace std;

void code(/* Copyright by Urtusea */)
{
    int x, y;
    cin >> x >> y;

    // 只需判断 y >= -1 即可
    if (y >= -1)
        cout << "YES\n";
    else
        cout << "NO\n";
}

int main(int argc, char *argv[], char *envp[])
{
    cin.tie(nullptr)->sync_with_stdio(false);

#ifdef Urtusea
    freopen("main.in",  "r", stdin );
    freopen("main.out", "w", stdout);
#endif

    for (int i = 1, n = (cin >> n, n); i <= n; i++)
    {
        code();
    }

    return 0;
}
