    D. a-Good String

    前言： #

    • Links: D. a-Good String
    • Codeforces: \(1500\)
    • Clist: \(1527\)
    • Tags: 分治，模拟，DFS
    • 时间复杂度: \(nlogn\)

    题意： #

    给你一个长度为 \(n\) 的字符串 \(s\) ，定义 \(char(s)\) 为 \(s\) 的字符集合大小，并给你一个 \(good(s, c)\) 串定义：

    \( good(s, c) = \begin{cases} if \ \ |s| = 1 \ and \ s_1 = c \\ if \ \ char(s_1s_2...s_{n/2}) = 1 \ and \ s_1 = c \ and \ good(s_{n/2+1}...s_n, c + 1) \\ if \ \ char(s_{n/2+1}...s_n) = 1 \ and \ s_n = c \ and \ good(s_1s_2...s_{n/2}, c + 1) \end{cases} \)

    现在你可以进行任意次操作，将 \(s\) 中的一个字符修改成另一个字符，请问将 \(s\) 修改成 \(good(s, a)\) 串的最小操作数

    思路： #

    模拟操作，不难得出将一个 \(s\) 修改成 \(good(s, c)\) 的操作数 \(f(s,c)\) 为：

    \( f(s,c) = min(\sum_{i = 1}^{l} + f(r,c+1), \ \sum_{i = 1}^{r} + f(l,c+1)) \)

    只需模拟操作即可

    代码： # 

    #include <bits/stdc++.h>
using namespace std;

void code(/* Copyright by Urtusea */)
{
    int n;
    string s;
    cin >> n >> s;

    // 个人习惯
    s = '#' + s;

    function<int(int, int, char)> dfs = [&](int u, int len, char c)->int
    {
        if (len == 1) return s[u] != c;

        // 递归左右区间
        int l = dfs(u,           len / 2, c + 1);
        int r = dfs(u + len / 2, len / 2, c + 1);

        // 统计剩余部分是否全为 c
        int cntL = len / 2 - count(s.begin() + u, s.begin() + u + len / 2, c);
        int cntR = len / 2 - count(s.begin() + u + len / 2, s.begin() + u + len, c);

        return min(l + cntR, r + cntL);
    };

    cout << dfs(1, n, 'a') << '\n';
}

int main(int argc, char *argv[], char *envp[])
{
    cin.tie(nullptr)->sync_with_stdio(false);

#ifdef Urtusea
    freopen("main.in",  "r", stdin );
    freopen("main.out", "w", stdout);
#endif

    for (int i = 1, n = (cin >> n, n); i <= n; i++)
    {
        code();
    }

    return 0;
}
