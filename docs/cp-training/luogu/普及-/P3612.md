# [Luogu P3612 - [USACO17JAN] Secret Cow Code S](https://www.luogu.com.cn/problem/P3612)

## Tag: 分治，递归，模拟

## 题意

按照题意输出对应位置的字符

## 思路

考虑往前递推即可

## 代码

``` cpp
#include <bits/stdc++.h>

using Int  = int64_t;
using uInt = uint64_t;

auto code() -> void {
    std::string s;
    std::cin >> s;

    Int N;
    std::cin >> N;

    while (N > s.size()) {
        Int tmp = s.size();
        while ((tmp << 1) < N)
            tmp <<= 1;
        N -= tmp;
        if (N == 1)
            N = tmp;
        else
            N = N - 1;
    }

    std::cout << s[N - 1] << '\n';
}

auto main(int argc, char *argv[], char *envp[]) -> int {
    std::cin.tie(nullptr)->sync_with_stdio(false);

    // for (int i = 1, n = (std::cin >> n, n); i <= n; ++i)
        code();

    return 0;
}
```
